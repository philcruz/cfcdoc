CFCDoc v1.0 (July 6, 2010)

INSTALLATION

1.  Unzip the files into your web root in a \cfdoc directory (for example, c:\inetpub\wwwroot\cfcdoc )

2.  If you install CFCDoc into <[cfusionmx|blueDragon_Server_JX_61]-webroot>/cfcdoc you will not need to create a mapping.
    If you install it anywhere else, create a mapping "cfcdoc" that points to the installation directory. 	

3.  Open the application in a browser.
