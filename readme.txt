CFCDoc

INSTALLATION

1.  Unzip the files into your web root (for example, c:\inetpub\wwwroot )
    The archive will extract to the directory \cfcdoc. 	

2.  Edit the <cfcdoc-home>/config/config.xml file.
    *  edit the  <licensekey>  line with your license key.

3.  If you install CFCDoc into <[cfusionmx|blueDragon_Server_JX_61]-webroot>/cfcdoc you will not need to create a mapping.
    If you install it anywhere else, create a mapping "cfcdoc" that points to the installation directory. 	

4.  Open the application in a browser. 

SYSTEM REQUIREMENTS

* Operating system: Windows (XP/2000/2003),Linux,Unix,Mac OS X
* Supported web servers: IIS 5, IIS 6, Apache
* Supported application servers: Macromedia CFMX 6.1, New Atlanta BlueDragon 6.1 Server JX
